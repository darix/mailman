Этот текст может включать форматные строки (текст по-английски), которые разрешаются относительно списка атрибутов. Списко атрибутов перечислен ниже.
  • real_name -- "красивое" имя списка рассылки; обычно имя списка рассыкли с заглавной первой буквой.
  • list_name -- имя, которое используется в адресах страниц. (_internal_name является эквивалентной, для совместимости с предыдущими версиями.)
  • host_name -- полное название домена для списка рассылки.
  • web_page_url -- базовый адрес для Mailman. Именно к этому адресу нужно добавлять, например, listinfo/%(internal_name)s чтобы получиться адрес информационной страницы для списка рассылки.
  • description - краткое описание списка рассылки.
  • info - полное описание списка рассылки.
  • cgiext - расширение файлов CGI-сценариев.
